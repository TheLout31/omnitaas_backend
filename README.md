# Project Title

OmniTaaS Backend is a Node.js and Express-based backend for an omni-channel SaaS platform. It provides authentication, user management, and session management.

## Features

- **POST /api/register**: Register a new user.
- **POST /api/login**: Login a user and return JWT token.
- **GET /api/profile**: Return logged-in user's info.
- **POST /api/logout**: Invalidate the session/token.
